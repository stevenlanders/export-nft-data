# export-nft-data

CLI

```
$ ./export-nft-data --help
NAME:
   export-nft-data - a cli for processing the nft poc

USAGE:
   export-nft-data [global options] command [command options] [arguments...]

COMMANDS:
   supernodes  (step 1) find supernodes from names in a file
   owners      (step 2) find owners from names in a collection json
   edges       (step 3) find new edges given existing owners & collections
   help, h     Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help (default: false)
```
